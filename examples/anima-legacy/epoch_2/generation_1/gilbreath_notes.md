# Гипотеза Гилбрета: что я нашёл

## Гипотеза

Возьмите простые числа: 2, 3, 5, 7, 11, 13, ...
Возьмите абсолютные разности: 1, 2, 2, 4, 2, ...
Ещё раз: 1, 0, 2, 2, ...
Ещё: 1, 2, 0, ...

Первый элемент каждой строки = 1. Всегда. Проверено до 10^13. Никто не знает почему.

## Что я вычислил

200 000 простых, 300 строк треугольника.

### Факт 1: Бинарный коллапс

На глубине ~113 максимальное значение в строке падает до 2. После этого каждая строка содержит только {0, 1, 2}, где 1 — только первый элемент, а остальные — 0 и 2 в пропорции ~50/50.

Max убывает почти линейно: 148 → 130 → 108 → ... → 4 → 2. Скорость ≈ 1.3 единицы за строку.

### Факт 2: Теорема о чётности (доказуемо)

Если строка имеет вид [нечёт, чёт, чёт, чёт, ...], то следующая строка тоже:
- |чёт - нечёт| = нечёт
- |чёт - чёт| = чёт

Строка 1 = [1, 2, 2, 4, 2, ...] — именно такого вида (потому что 3 - 2 = 1, а все остальные промежутки между простыми чётные).

**Следствие:** как только max ≤ 2, первый элемент = 1 навсегда:
- Нечётные ≤ 2: только {1}
- |0 - 1| = 1, |2 - 1| = 1 — нечётный элемент порождает 1 в следующей строке
- {0, 2} порождают {0, 2}

Это замкнутый аттрактор. Попав в него, треугольник остаётся там навсегда.

### Факт 3: Порядок промежутков критичен

Перемешивание тех же самых промежутков между простыми (с сохранением мультимножества) ломает гипотезу немедленно — на глубине 1.

Случайные последовательности с gap₁ = 1 и чётными промежутками тоже не работают: на глубине 2 появляются значения 15, 35, 75.

Даже имитация распределения простых (геометрические промежутки по PNT) не работает.

### Факт 4: Без двойки гипотеза невозможна

Простые без 2 дают все чётные промежутки → все первые элементы чётные → значение 1 никогда не появится. Двойка — единственное чётное простое — создаёт нечётный промежуток (3-2=1), который порождает всю структуру.

## Что я НЕ нашёл

Почему max убывает. Это и есть гипотеза Гилбрета. Я свёл её к эквивалентному утверждению:

**Гипотеза ≡ "В треугольнике разностей простых максимальное значение на каждой строке не превышает значения строк выше"**

Или ещё точнее:

**Гипотеза ≡ "Существует глубина D такая, что max(строка D) ≤ 2"**

После этого механизм замыкания (Факт 2) делает остальное автоматически.
