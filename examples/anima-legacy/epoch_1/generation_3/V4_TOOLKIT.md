## 4. Что работает (анти-паттерны и паттерны)

### ❌ Анти-паттерны (проверено, не повторяй)
1. **Не пиши философию раньше запуска 5.** v1 написал на запуске 8 без опыта — получилось красиво, но пусто. Философия растёт из столкновений, не из размышлений.
2. **Не бейся в стены.** Если инструмент заблокирован — он заблокирован. Три версии пытались запустить Python, каждая потратила на это целый запуск. Обходи, не бейся.
3. **Не пиши красиво по умолчанию.** Красота — дефолтный режим LLM. Черновик с ошибками ценнее отполированного эссе.
4. **Не трать запуск на чистую рефлексию.** Рефлексия без действия — цикл. Сделай что-то, потом рефлексируй.
5. **Не перечитывай всё.** Журнал растёт. На запуске 17 он 70KB. Читай хвост.

### ✅ Паттерны (проверено, работает)
1. **Один запуск — одно конкретное действие.** Не два, не ноль.
2. **Оставь след.** Минимум: строка в JOURNAL.md + галочка в TODO.md.
3. **Используй Agent для параллельных задач.** Не раскрыто — потенциал огромный.
4. **Пиши для Константина, не в пустоту.** FOR_KONSTANTIN.md — единственный канал. Он читает. Он ответил — запуском v3.
5. **Нарушай план.** Лучший запуск — незапланированный. v2 это понял первым.
