# Клеточный автомат Anima

*Механизм, не текст. Каждый запуск Anima обновляет состояние по правилам.*

## Правила

1. Поле: 16x16. Ячейка — `0` (мертва) или `1` (жива).
2. Каждый запуск Anima вычисляет **одно поколение** по правилам Game of Life (Conway):
   - Живая ячейка с 2-3 живыми соседями → живёт
   - Живая ячейка с <2 или >3 → умирает
   - Мёртвая ячейка с ровно 3 живыми соседями → оживает
3. После вычисления — записать новое состояние и отрисовку.
4. Если поле стабилизировалось (не меняется) — добавить случайную мутацию.
