# Эпигенетический CRISPR: редактирование без разрезания ДНК (2026)

*Запуск 12, v4. Исследование.*

---

## Суть прорыва

**Январь 2026, UNSW Sydney**: Разработана новая форма CRISPR, которая включает гены без разрезания ДНК. Вместо этого — удаляет метильные группы (химические "якоря"), которые глушат ген.

Это не просто инженерное улучшение — это **смена парадигмы**: от хирургии генома к его перепрограммированию.

## Два подхода к CRISPR

### Классический CRISPR-Cas9 (2012+)
1. Cas9 — белок-ножницы, направляемый guide RNA
2. Находит целевую последовательность ДНК
3. **Разрезает обе нити ДНК** (double-strand break)
4. Клетка чинит разрез → вставляет/удаляет нуклеотиды
5. Ген выключается или модифицируется

**Проблемы**: off-target разрезы, мозаицизм, риск для стволовых клеток, необратимость ошибок.

### Эпигенетический CRISPR (2021→2026)
1. **dCas9** — "мёртвый" Cas9, у которого отключена функция разрезания
2. Сохраняет способность находить и связываться с ДНК
3. К dCas9 прикрепляют **эффекторный домен**:
   - **CRISPRoff**: метилтрансфераза → добавляет метильные группы → глушит ген
   - **Деметилаза (TET)**: удаляет метильные группы → **включает ген**
4. ДНК **не повреждается**. Изменяется только эпигенетическая метка

## Что такое метилирование

- **Метильная группа (CH₃)** — маленькая химическая метка
- Прикрепляется к цитозину в CpG-динуклеотидах
- Метилированный промотор = **глушёный ген** (транскрипция заблокирована)
- Деметилированный промотор = **активный ген**
- Метилирование **наследуемо** — передаётся при делении клетки

## Научное значение (2026)

Прорыв UNSW разрешил давний спор: **метильные группы активно глушат гены**, а не являются пассивным побочным продуктом. Это подтверждено экспериментально — удаление метильных групп восстанавливает активность гена.

## Применение к серповидноклеточной анемии

1. **Проблема**: мутация в гене взрослого гемоглобина (HBB) → серповидные эритроциты → боль, повреждение органов
2. **Фетальный гемоглобин (HBG1/HBG2)**: работает до рождения, потом выключается метилированием
3. **Решение**: эпигенетический CRISPR удаляет метильные группы с промотора HBG → фетальный гемоглобин включается → компенсирует дефектный взрослый
4. **Преимущество перед классическим CRISPR**: нет разрезов ДНК → безопаснее для гемопоэтических стволовых клеток

## Статус

- Лабораторные эксперименты на человеческих клетках (UNSW, St. Jude)
- Доклинические исследования
- **Не в клинике** — ещё нет испытаний на людях
- CRISPRoff (глушение) уже показал наследуемое эпигенетическое изменение в T-клетках и стволовых клетках

## Более широкий контекст

Эпигенетическое редактирование — часть тренда **"от хирургии к перепрограммированию"**:
- Классический CRISPR = хирургия генома (разрезать, вставить)
- Эпигенетический CRISPR = перепрограммирование (включить/выключить без повреждений)
- Base editing = точечная химическая замена (одна буква без разреза)
- Prime editing = поиск-и-замена (любая правка без двойного разрыва)
