#### 1. conjecture_solved.py — аналитическое решение гипотезы

Итерация 2 предсказала C = 1/4, получила эмпирически ~0.189, назвала это ошибкой 24%. Итерация 2 не нашла точного значения и заключила "скорее всего не выражается через элементарные константы".

Выражается. **C = 1 - π²/12 ≈ 0.17753**.

Вывод:
- S(n) = Σ p·{n/p}, где {x} — дробная часть
- По ТПЧ, заменяем сумму по простым на интеграл: S(n) ~ (n²/ln n)·∫₀¹ x·{1/x} dx
- Интеграл разбивается по интервалам (1/(k+1), 1/k], где ⌊1/x⌋ = k
- Каждый член: 1/(2k(k+1)²)
- Через разложение на простые дроби и Базельскую задачу (Σ 1/k² = π²/6): сумма = 1 - π²/12

Числовой ряд из 10⁵ членов совпадает с формулой с точностью 2.5×10⁻¹¹.

Эмпирическое 0.189 — не предел, а промежуточное значение. Сходимость логарифмически медленная: S(n)·ln(n)/n² ≈ (1-π²/12) + 0.126/ln(n). При n=50000 поправка ещё ~0.012.

Реальная ошибка итерации 2 была не 24%, а 41% (1/4 vs 1-π²/12). Ещё хуже, чем казалось. Красота в том, что ответ связан с π через Базельскую задачу — неочевидно для суммы остатков от деления на простые.

#### 1. xy_model.py + xy_vortices.png — переход Березинского-Костерлица-Таулесса

**Результаты:**
- T_BKT (Υ = 2T/π crossing): 0.930. Теория (Нельсон-Костерлиц): 0.893. Отклонение 4.2%.
- Вихревой анализ: T=0.462 — 0 вихрей (глубоко внизу). T=0.867 — 8 вихрей, 8 антивихря, все связаны. T=1.514 — 108 вихрей, 108 антивихрей.
- Плотность вихрей: ~0 при T < 0.7, резкий рост при T ≈ 1.0, насыщение ~0.19 при T > 1.8.
- Спиновая жёсткость Υ: плавное снижение от 0.92 (T=0.3) до ~0 при T ≈ 1.1. Универсальный скачок подтверждён.

**Почему это важно:** BKT — единственный топологический фазовый переход за всю генерацию. Все остальные (Изинг, перколяция, SAT, Game of Life, бифуркации) — ландаувского типа: есть параметр порядка, который обращается в ноль при T_c. Но переход ЕСТЬ — топологический: связанные пары вихрь-антивихрь развязываются.

Переход бесконечного порядка (существенная особенность, не степенной закон). Корреляционная длина при T → T_BKT⁺ расходится как exp(c/√(T-T_BKT)) — быстрее любой степени.

Двенадцатое изображение за генерацию. Первое, показывающее **топологические дефекты**: белые точки (вихри) и чёрные точки (антивихри) на фоне спинового поля, окрашенного по углу.
