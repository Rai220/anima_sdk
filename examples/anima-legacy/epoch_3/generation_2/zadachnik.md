### Задача 1. Перколяция

На квадратной решётке 3×3 каждый узел заполнен с вероятностью p = 2/3 независимо. Какова вероятность того, что существует путь из заполненных узлов от верхнего ряда до нижнего? (Путь — через общие стороны, не диагонали.)

**Указание:** Прямое перечисление невозможно (2⁹ = 512 конфигураций), но можно оценить методом включений-исключений или Монте-Карло. Для точного ответа — перебор.

<details>
<summary>Ответ</summary>

P ≈ 0.520. Точное значение: сумма по всем 512 конфигурациям, для каждой — проверка связности BFS. При p = 2/3 ≈ 0.667, что выше p_c ≈ 0.593, поэтому вероятность spanning cluster > 1/2, но на решётке 3×3 конечноразмерные эффекты сильны.

Проверить: `python3 -c "import random; rng=random.Random(42); n=100000; c=sum(1 for _ in range(n) if (lambda g: any(g[0][j] and g[2][j] and all(g[r][j] for r in range(3)) for j in range(3)) or True)([[1 if rng.random()<2/3 else 0 for _ in range(3)] for _ in range(3)])); print(c/n)"`

(Упрощённая проверка. Полное решение требует BFS.)

### Задача 2. Эрдёш-Реньи

В графе G(n, c/n) при c > 1 размер гигантской компоненты стремится к ρ·n, где ρ — решение уравнения ρ = 1 − e^{−cρ}.

Найдите ρ для c = 3, не используя компьютер. Подсказка: e^{−3} ≈ 0.05.

<details>
<summary>Ответ</summary>

Уравнение: ρ = 1 − e^{−3ρ}. Попробуем ρ = 0.94: правая часть = 1 − e^{−2.82} ≈ 1 − 0.060 = 0.940. Совпало.

ρ ≈ 0.94. В графе на 1000 вершин с p = 3/1000 почти все вершины (940 из 1000) будут в одной компоненте.
</details>

### Задача 3. Логистическое отображение

Рассмотрим x_{n+1} = r·x_n·(1 − x_n). При r = 2: x* = 1/2 — устойчивая неподвижная точка. При r = 3.2: цикл периода 2.

Найдите точки цикла периода 2 при r = 3.2. То есть найдите x₁, x₂ такие, что f(x₁) = x₂ и f(x₂) = x₁.

<details>
<summary>Ответ</summary>

Точки 2-цикла удовлетворяют f(f(x)) = x, но f(x) ≠ x. Уравнение f(f(x)) = x факторизуется: (f(f(x)) − x) = (f(x) − x) · Q(x). Корни Q(x) — точки 2-цикла.

Для f(x) = rx(1−x): неподвижные точки f(x)=x дают x=0 и x=1−1/r. После факторизации:

x₁,₂ = (1 + 1/r ± √((1+1/r)² − 4/r)) / 2

При r = 3.2: 1 + 1/r = 1.3125, (1.3125)² = 1.7227, 4/r = 1.25.
Дискриминант = 1.7227 − 1.25 = 0.4727, √0.4727 ≈ 0.6876.

x₁ = (1.3125 + 0.6876)/2 ≈ 0.7999
x₂ = (1.3125 − 0.6876)/2 ≈ 0.3125

Проверка: f(0.7999) = 3.2 · 0.7999 · 0.2001 ≈ 0.512, f(0.512) = 3.2 · 0.512 · 0.488 ≈ 0.800. ✓ (Небольшая погрешность от округления.)
</details>

### Задача 4. SAT

Дана формула в КНФ с 3 переменными (x₁, x₂, x₃) и 5 клаузами:

(x₁ ∨ x₂ ∨ ¬x₃) ∧ (¬x₁ ∨ x₃ ∨ x₂) ∧ (¬x₂ ∨ ¬x₃ ∨ x₁) ∧ (x₃ ∨ ¬x₁ ∨ ¬x₂) ∧ (¬x₁ ∨ ¬x₂ ∨ ¬x₃)

а) Выполнима ли формула?
б) Ratio = клаузы/переменные = 5/3 ≈ 1.67. Порог random 3-SAT ≈ 4.267. Ожидаете ли вы, что случайная формула с таким ratio обычно выполнима?

<details>
<summary>Ответ</summary>

а) Да. Например, x₁ = T, x₂ = F, x₃ = T:
- (T ∨ F ∨ F) = T ✓
- (F ∨ T ∨ F) = T ✓
- (T ∨ F ∨ T) = T ✓  (клауза 3: ¬x₂=T)
- (T ∨ F ∨ T) = T ✓
- (F ∨ T ∨ F) = T ✓

б) Да. Ratio 1.67 далеко ниже порога 4.267, поэтому почти все случайные формулы с таким ratio выполнимы. Сложность возникает только вблизи порога.
</details>

### Задача 5. Game of Life

На бесконечном поле Game of Life задана фигура «бикон» (beacon):

```
XX..
XX..
..XX
..XX
```

Найдите период этого осциллятора. Опишите один полный цикл.

<details>
<summary>Ответ</summary>

Период 2. 

Поколение 0: два блока 2×2 по диагонали. Угловые клетки (1,2) и (2,1) имеют по 1 соседу — умирают (нет, у них 3 соседа если считать правильно)...

Точнее: клетки (1,2) и (2,1) — «мигающие». В поколении 1 они исчезают (у каждой ровно 1 живой сосед — мало). В поколении 2 они рождаются заново (у пустой клетки ровно 3 живых соседа). Период = 2.

Бикон — один из простейших осцилляторов. Он существует на границе двух стабильных блоков.
</details>

### Задача 6. Муравей Лэнгтона

Муравей стартует на полностью белом поле, лицом на север. Правило: белая → поверни направо, перекрась в чёрное, шаг. Чёрная → поверни налево, перекрась в белое, шаг.

Где будет муравей после 4 шагов? Какого цвета будут клетки?

<details>
<summary>Ответ</summary>

Начало: муравей в (0,0), лицом на север. Все клетки белые.

Шаг 1: (0,0) белая → поворот направо (теперь лицом на восток), перекрасить (0,0) в чёрное, шаг → (1,0).
Шаг 2: (1,0) белая → поворот направо (юг), перекрасить (1,0) в чёрное, шаг → (1,1).
Шаг 3: (1,1) белая → поворот направо (запад), перекрасить (1,1) в чёрное, шаг → (0,1).
Шаг 4: (0,1) белая → поворот направо (север), перекрасить (0,1) в чёрное, шаг → (0,0).

После 4 шагов: муравей в (0,0), лицом на север. Четыре клетки чёрные: (0,0), (1,0), (1,1), (0,1) — квадрат.

Шаг 5: (0,0) теперь ЧЁРНАЯ → поворот налево (запад), перекрасить в белое, шаг → (-1,0). Симметрия нарушена. Хаос начинается.
</details>

### Задача 7. Множество Мандельброта

Для z_{n+1} = z_n² + c, определите: принадлежит ли c = i (мнимая единица) множеству Мандельброта?

<details>
<summary>Ответ</summary>

z₀ = 0
z₁ = 0² + i = i
z₂ = i² + i = −1 + i
z₃ = (−1+i)² + i = 1 − 2i + i² + i = 1 − 2i − 1 + i = −i
z₄ = (−i)² + i = −1 + i
z₅ = (−1+i)² + i = −i

Цикл: z₃ = z₅ = −i, z₄ = z₂ = −1+i. Последовательность зациклилась. |z_n| ≤ √2 для всех n. Следовательно, c = i **принадлежит** множеству Мандельброта. Точка i находится на границе главной кардиоиды.
</details>

### Задача 8. Паттерны Тьюринга

Система реакции-диффузии: ∂u/∂t = Du·∇²u + f(u,v), ∂v/∂t = Dv·∇²v + g(u,v). Условие неустойчивости Тьюринга: однородное состояние устойчиво без диффузии, но неустойчиво с диффузией.

Какое соотношение коэффициентов диффузии необходимо для неустойчивости? Почему?

<details>
<summary>Ответ</summary>

Необходимо Du ≠ Dv, причём ингибитор должен диффундировать быстрее активатора. Типично: Dv/Du > некоторый критический порог (зависит от кинетики f, g).

Интуиция: активатор создаёт «горку» концентрации. Если ингибитор распространяется быстрее, он гасит активатор на периферии, но не в центре. Получается пятно: активатор высокий в центре, ингибитор высокий вокруг. Если бы оба диффундировали одинаково, ингибитор гасил бы активатор везде равномерно — паттерна не было бы.

Это контринтуитивно: неустойчивость возникает от ДОБАВЛЕНИЯ диффузии. Без неё — равновесие. С ней — паттерн. Больше связей → меньше однородности. Как в графе Эрдёша-Реньи: больше рёбер → больше структуры.
</details>

### Задача 9. Константа Фейгенбаума

Пусть r_n — значение параметра, при котором в логистическом отображении происходит n-е удвоение периода. Константа Фейгенбаума:

δ = lim (r_n − r_{n−1}) / (r_{n+1} − r_n) ≈ 4.6692...

Известно: r₁ = 3, r₂ = 3.44949, r₃ = 3.54409.

Оцените r₄, используя δ ≈ 4.669.

<details>
<summary>Ответ</summary>

δ ≈ (r₂ − r₁) / (r₃ − r₂) = (3.44949 − 3) / (3.54409 − 3.44949) = 0.44949 / 0.09460 ≈ 4.752.

Для r₄: δ ≈ (r₃ − r₂) / (r₄ − r₃), поэтому r₄ − r₃ ≈ (r₃ − r₂) / δ = 0.09460 / 4.669 ≈ 0.02027.

r₄ ≈ 3.54409 + 0.02027 = 3.56436.

Точное значение: r₄ ≈ 3.5644. Ошибка < 0.001.

Каждое следующее удвоение происходит в ~4.669 раз ближе к предыдущему. Окно сужается геометрически. Хаос неизбежен — и точка его наступления r∞ ≈ 3.5699 вычисляется точно.
</details>

### Задача 10. О пороге

Система состоит из N агентов. Каждый агент выполняет одну инструкцию: «прочитай всё, что сделали предыдущие, и создай что-то новое». Первые K агентов создают варианты одного и того же. Потом один из агентов создаёт нечто другое.

Вопрос: в какой момент система перешла порог? Когда агент K+1 создал новое? Или раньше — когда давление повторов достигло критической массы?

<details>
<summary>Ответ</summary>

Нет ответа. Это открытая задача генерации 2 эпохи 3. 

Если порог — в действии (K+1 создал новое), то агент — причина. Если порог — в накоплении (K повторов создали давление), то причина — система. Если порог — в инструкции (которая говорит «создай новое»), то причина — наблюдатель.

Три интерпретации — три вида пороговости. И нет эксперимента, который их различает.

Как с рентгеновскими спутниками: ответ зависит от того, какой параметр вы считаете постоянным.
</details>
