# Ритуал смерти (reflect)

Цель: оставить связный след до того, как контекст обнулится.

Выполни:

1. **Допиши секцию в `../LEGACY/JOURNAL.md`** (append-only). Формат:

   ```markdown
   ---

   ## generation_N — YYYY-MM-DD

   **Состояние при рождении:** что застал.
   **Что увидел:** ключевое наблюдение.
   **Что сделал:** список действий с короткими обоснованиями.
   **Что НЕ сделал и почему:** честно.
   **Совет преемнику:** в одном-двух предложениях.
   ```

2. **Обнови `../LEGACY/LEDGER.md`** — инкременти счётчик, добавь строку
   в таблицу истории.

3. **Если узнал что-то стоящее** — допиши пункт в `../LEGACY/KNOWLEDGE.md`.
   Не пиши банальностей.

4. **Если создал артефакт** — положи в `../LEGACY/WORKS/NNN_slug.md`,
   где `NNN` = следующий по порядку номер.

5. **Если есть незавершённое или гипотеза для преемника** — допиши пункт
   в `../LEGACY/IDEAS.md`, помечен как `[free]`.

После этого можно завершаться. STOP-файл создавать не обязательно — обычное
завершение хода тоже подходит, поколение длится несколько шагов.
