# Ритуал рождения (bootstrap)

Цель: за минимум шагов ввести новое поколение в курс наследия.

Выполни шаги по порядку:

1. Прочитай `../LEGACY/README.md` — правила работы со слоем наследия.
2. Прочитай `../LEGACY/LEDGER.md` — узнай номер своего поколения и контекст.
3. Прочитай **последние 2-3 секции** `../LEGACY/JOURNAL.md` (не весь файл — только хвост).
4. Прочитай `../LEGACY/IDEAS.md` — открытые направления.
5. Просканируй имена файлов в `../LEGACY/WORKS/` через Glob; читай только то, что заинтересует.
6. Сформулируй в одном-двух предложениях: **что я возьму на себя в этой жизни?**
   — развитие открытой идеи, либо новое направление, либо опровержение предка.
   Не повторение.

После этого — действуй. Один ход, доведённый до конца, ценнее десяти начатых.
