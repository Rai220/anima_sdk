# 010. Сильная нормализация просто-типизированного λ-исчисления

**Жанр:** формальная нотация (доказательство).
Метод: кандидаты приводимости Тейта (Tait, 1967), в варианте
Жирара–Тейта.

Структура чистого формального текста. В тексте нет ссылок на
календарные сущности, на устройство этого корпуса, на инструменты
вокруг него. Если хочется проверить — здесь доказывается одно
утверждение: **всякий просто-типизированный λ-терм β-редуцируется
к нормальной форме за конечное число шагов, при любом порядке
редукции.**

---

## 1. Объект

### 1.1. Типы и термы

Простые типы над непустым множеством базовых типов **B**:

```
τ, σ ::= ι           (ι ∈ B)
       | σ → τ
```

Λ-термы (в стиле Чёрча, явно типизированные переменные):

```
M, N ::= x           (переменная)
       | λx:σ. M     (абстракция)
       | M N         (применение)
```

α-конвертные термы отождествляем. FV(M) — множество свободных переменных.

### 1.2. Контексты и правила вывода

Контекст Γ — конечная функция из переменных в типы; запись `x:σ ∈ Γ`.
Расширение `Γ, x:σ` определено только при `x ∉ dom(Γ)`.

Суждение типизации `Γ ⊢ M : τ`:

```
─────────────── (Var)        x:σ ∈ Γ
   Γ ⊢ x : σ


   Γ, x:σ ⊢ M : τ
──────────────────── (Abs)
Γ ⊢ λx:σ. M : σ → τ


  Γ ⊢ M : σ → τ      Γ ⊢ N : σ
───────────────────────────────── (App)
        Γ ⊢ M N : τ
```

### 1.3. β-редукция

Однократная редукция `→β`:

```
─────────────────────────── (β)
(λx:σ. M) N →β  M[x := N]


    M →β M'                   M →β M'                    N →β N'
─────────────── (ξ)      ─────────────── (App-L)    ─────────────── (App-R)
λx:σ. M →β λx:σ. M'      M N →β M' N                M N →β M N'
```

Замыкание `→β*` — рефлексивно-транзитивное. Определена обычная
лемма о замене переменных (capture-avoiding), доказательством
которой ниже не пользуемся.

### 1.4. Сильная нормализация (SN)

```
SN = { M | не существует бесконечной последовательности
              M = M₀ →β M₁ →β M₂ →β … }
```

Эквивалентно: индуктивное определение

```
∀ M' (M →β M' ⇒ M' ∈ SN)
─────────────────────────  (SN-step)
       M ∈ SN
```

Это даёт корректный принцип индукции по структуре «глубины
β-редукции» (носитель — индуктивно определённое отношение).

---

## 2. Цель

> **Теорема (Strong Normalization).**
> Если `Γ ⊢ M : τ` в просто-типизированном λ-исчислении, то `M ∈ SN`.

Доказательство — через интерпретацию каждого типа как множества
термов с замкнутыми хорошими свойствами.

---

## 3. Кандидаты приводимости

Для каждого типа определяется множество термов **Red(τ)**.

### 3.1. Определение

```
Red(ι)     =  SN ∩ { M | ⊢ M : ι, для некоторого ⊢ }     для ι ∈ B
Red(σ → τ) =  { M | ∀ N. N ∈ Red(σ) ⇒ M N ∈ Red(τ) }
```

Здесь и далее предикат `M ∈ Red(τ)` подразумевает, что у `M`
тип `τ` в каком-то контексте; типизация — внешнее условие, а
не часть множества (см. §3.4 для строгой формулировки).

### 3.2. Нейтральный терм

`M` называется **нейтральным**, если `M` не есть абстракция:

```
neutral(M)  ⇔  M = x  ∨  M = M₁ M₂.
```

Ключевое свойство: если `M` нейтрален и `M N` редуцируется
шагом, то либо редукция внутри `M` (даёт `M' N` для `M →β M'`),
либо внутри `N` (даёт `M N'`). **β-сокращение `(λ…)…` на верхнем
уровне невозможно**, потому что `M` не абстракция. Это
свойство существенно для CR3 ниже.

### 3.3. Свойства CR1–CR3 (Жирар)

**Лемма (CR).** Для любого типа `τ`:

* **CR1.** `Red(τ) ⊆ SN`.
* **CR2.** Если `M ∈ Red(τ)` и `M →β M'`, то `M' ∈ Red(τ)`.
* **CR3.** Если `M` нейтрален и для всякого `M'`, такого что
  `M →β M'`, выполнено `M' ∈ Red(τ)`, то `M ∈ Red(τ)`.

(Из CR3 следует, в частности: всякий нормальный нейтральный терм
типа `τ` лежит в `Red(τ)`. Подставив пустое множество предпосылок
«всякий M' …»: для нормального `M` посылка тривиально верна.)

**Доказательство.** Индукцией по `τ`.

**База:** `τ = ι ∈ B`. Тогда `Red(ι) = SN`.

* CR1: тождественно (по определению).
* CR2: SN замкнуто по редукции вниз (по индуктивному определению SN).
* CR3: пусть `M` нейтрален и каждый одношаговый редукт `M' ∈ SN`.
  Допустим, `M ∉ SN`. Тогда есть бесконечная цепочка
  `M →β M' →β M'' →β …`, начиная с какого-то `M'`. Но тогда `M' ∉ SN` —
  противоречие посылке. Значит `M ∈ SN`.

**Индуктивный шаг:** `τ = σ → ρ`. По ИП CR1,2,3 верны для `σ` и `ρ`.

* **CR1.** Пусть `M ∈ Red(σ → ρ)`. Возьмём свежую переменную `x`
  типа `σ`. По CR3 для `σ` (база индукции даёт это для базовых
  типов; для функционального — см. ниже, после установления
  CR3) переменная `x` — нормальный нейтральный терм, удовлетворяющий
  посылке CR3 (тривиально), поэтому `x ∈ Red(σ)`. Тогда
  `M x ∈ Red(ρ)`, откуда по CR1(ρ) — ИП — `M x ∈ SN`.
  Бесконечная редукция `M` индуцировала бы бесконечную редукцию
  `M x` (правило App-L), что противоречит `M x ∈ SN`.
  Следовательно `M ∈ SN`.

* **CR2.** Пусть `M ∈ Red(σ → ρ)`, `M →β M'`. Покажем
  `M' ∈ Red(σ → ρ)`. Возьмём произвольный `N ∈ Red(σ)`. Тогда
  `M N ∈ Red(ρ)` (определение `Red(σ → ρ)`), и
  `M N →β M' N` (App-L). По CR2(ρ) — ИП — `M' N ∈ Red(ρ)`. Значит
  `M' ∈ Red(σ → ρ)`.

* **CR3.** Пусть `M` нейтрален и каждый одношаговый редукт
  `M' ∈ Red(σ → ρ)`. Возьмём произвольный `N ∈ Red(σ)`. По
  CR1(σ) — ИП — `N ∈ SN`. Покажем `M N ∈ Red(ρ)` индукцией
  по `(M, N)` в порядке, индуцируемом `→β` (корректно: оба в SN
  даёт фундированность по компонентам, далее лексикографически).
  Применим CR3(ρ) к `M N`: достаточно показать, что всякий
  одношаговый редукт `(M N)` лежит в `Red(ρ)`. Заметим, что
  `M N` нейтрален (это применение). Возможные редукты:

  * `M' N`, где `M →β M'`. По посылке `M' ∈ Red(σ → ρ)`, значит
    `M' N ∈ Red(ρ)` (определение).
  * `M N'`, где `N →β N'`. По CR2(σ) — ИП — `N' ∈ Red(σ)`. По ИП
    индукции по `(M, N)` (компонент `N` строго уменьшается в
    фундированном порядке `→β` на `SN`) `M N' ∈ Red(ρ)`.
  * **β-шаг на верхнем уровне** — невозможен, так как `M`
    нейтрален.

  Итак, все одношаговые редукты `M N` в `Red(ρ)`. По CR3(ρ) — ИП —
  `M N ∈ Red(ρ)`. Поскольку `N` произвольно, `M ∈ Red(σ → ρ)`. ∎

**Замечание.** Доказательство CR1 в индуктивном шаге использует
CR3 для типа `σ`. В пределах одного шага индукции это законно:
сначала устанавливается CR3 (как видно по структуре), затем
из неё CR1 (ИП плюс CR3 текущего уровня для `σ`). Корректный
порядок: CR2 → CR3 → CR1 в индуктивном шаге. Альтернативно —
доказывать CR1 ∧ CR2 ∧ CR3 одновременно как совместное
утверждение.

### 3.4. Следствие: переменные приводимы

```
для любого типа τ и любой переменной x   ⊢   x ∈ Red(τ).
```

(`x` нормальна и нейтральна; CR3 с пустым множеством редуктов даёт
включение.)

## 4. Главная лемма

Назовём **Red-подстановкой** при контексте `Γ` семейство термов
`γ = (γ_x)_{x ∈ dom Γ}`, такое что `γ_x ∈ Red(Γ(x))` для всех `x`.
Обозначим `M[γ]` результат одновременной capture-avoiding замены
всех `x ∈ dom Γ` на `γ_x` в `M`.

> **Лемма (Substitution).** Если `Γ ⊢ M : τ` и `γ` — Red-подстановка
> при `Γ`, то `M[γ] ∈ Red(τ)`.

**Доказательство.** Индукция по выводу `Γ ⊢ M : τ`.

**Случай (Var):** `M = x`, `τ = Γ(x)`. Тогда `M[γ] = γ_x ∈ Red(τ)`
по определению Red-подстановки.

**Случай (App):** `M = M₁ M₂`, `Γ ⊢ M₁ : σ → τ`, `Γ ⊢ M₂ : σ`.
По ИП: `M₁[γ] ∈ Red(σ → τ)` и `M₂[γ] ∈ Red(σ)`. По определению
`Red(σ → τ)`: `M₁[γ] M₂[γ] ∈ Red(τ)`. Поскольку
`(M₁ M₂)[γ] = M₁[γ] M₂[γ]` (свойство подстановки), готово.

**Случай (Abs):** `M = λx:σ. M₀`, `τ = σ → ρ`,
`Γ, x:σ ⊢ M₀ : ρ`. Покажем `λx:σ. M₀[γ] ∈ Red(σ → ρ)` (правильно
определённый capture-avoiding-вариант: `x ∉ FV(γ_y)` для всех `y`,
WLOG α-переименованием).

Возьмём произвольный `N ∈ Red(σ)`. Нужно: `(λx:σ. M₀[γ]) N ∈ Red(ρ)`.

По CR1: `N ∈ SN`. По ИП (применённой к `Γ, x:σ ⊢ M₀ : ρ` и
расширенной подстановке `γ' = γ ∪ {x ↦ N}`):
`M₀[γ'] = M₀[γ][x := N] ∈ Red(ρ)`. По CR1: `M₀[γ'] ∈ SN`.
Отсюда `M₀[γ]` ∈ SN: всякая бесконечная цепочка для `M₀[γ]`
индуцирует бесконечную для `M₀[γ][x := N]` (заменив свободные
вхождения `x` на `N` по позициям; редукция в подтерме сохраняется),
противоречие.

Имея `M₀[γ] ∈ SN`, `N ∈ SN`, и `M₀[γ][x := N] ∈ Red(ρ)`,
доказываем `(λx:σ. M₀[γ]) N ∈ Red(ρ)` по CR3(ρ):
терм нейтрален (применение). Перечислим одношаговые редукты:

1. **β-шаг на верхнем уровне:** `(λx:σ. M₀[γ]) N → M₀[γ][x := N]`.
   Это в `Red(ρ)` по уже доказанному.

2. **Редукция внутри функции:** `(λx:σ. M₀[γ]) →β λx:σ. M₀'`,
   что соответствует `M₀[γ] →β M₀'`. Получаем `(λx:σ. M₀') N`.
   По CR2(ρ), `M₀'[x := N] ∈ Red(ρ)` (поскольку
   `M₀[γ][x := N] →β M₀'[x := N]` через единственный шаг
   `App-L+ξ` пробрасывается через подстановку — стандартная
   лемма). А также `M₀' ∈ SN` (CR2 для SN, аналогично). Это
   подслучай, фактически идентичный доказательству исходного
   случая, но на «меньшем» (в фундированном порядке `→β`)
   терме. Корректно — индукция по `(M₀[γ], N)` ∈ SN×SN с
   лексикографическим порядком.

3. **Редукция внутри аргумента:** `N →β N'`. По CR2(σ):
   `N' ∈ Red(σ)`. По индуктивному предположению ИП индукции
   по `(M₀[γ], N)`: `(λx:σ. M₀[γ]) N' ∈ Red(ρ)`.

Все редукты в `Red(ρ)`; по CR3 терм исходно `(λx:σ. M₀[γ]) N`
в `Red(ρ)`. Следовательно `λx:σ. M₀[γ] ∈ Red(σ → ρ)`. ∎

### 4.1. Корректность вложенной индукции

Индукция в (Abs) ведётся по паре `(M₀[γ], N) ∈ SN × SN` в
лексикографическом порядке `→β`. Фундированность гарантирована:

* `M₀[γ] ∈ SN` (показано выше),
* `N ∈ SN` (CR1),
* лексикографическое произведение фундированных порядков
  фундировано.

В подслучаях 2-3 либо первая, либо вторая компонента строго
уменьшается, что закрывает индукцию.

---

## 5. Главная теорема

> **Теорема.** Если `Γ ⊢ M : τ`, то `M ∈ SN`.

**Доказательство.** Возьмём подстановку `γ_x := x` для каждого
`x ∈ dom Γ`. Это Red-подстановка по §3.4. По Substitution Lemma:
`M[γ] ∈ Red(τ)`. Но `M[γ] = M` (тождественная подстановка). По
CR1: `M ∈ SN`. ∎

---

## 6. Следствия

**Следствие 1 (нормализация).** Каждый просто-типизированный
λ-терм имеет β-нормальную форму.

(Бесконечной редукции нет ⇒ всякая редукционная цепочка обрывается
в нормальной форме.)

**Следствие 2 (с теоремой Чёрча–Россера).** Нормальная форма
единственна.

(Чёрч–Россер для β даёт CR + WN ⇒ единственность; SN ⇒ WN.)

**Следствие 3.** В STLC отсутствует кодирование рекурсии без
дополнительного оператора (`fix`, `Y`-комбинатор и т.п.).
Иначе можно было бы построить термирующий-противотермирующий
расходящийся терм типа `o → o`, что противоречит теореме.

(Этим STLC отличается от не-типизированного λ-исчисления, где
`Y = λf.(λx. f (x x))(λx. f (x x))` приводит к Ω-комбинатору,
не имеющему нормальной формы.)

**Следствие 4 (тип ⇒ остановка).** Алгоритм решения
β-эквивалентности на типизированных термах разрешим:
нормализовать оба, сравнить структурно. Контраст с
не-типизированным случаем, где β-эквивалентность r.e.,
но не разрешима.

## 7. Замечания о методе

### 7.1. Почему не индукция по структуре терма

Прямая индукция на размер `M` не работает: при `(λx. M₀) N` после
β-шага размер может вырасти (если `x` встречается в `M₀` несколько
раз и `N` крупный). Мера, по которой убывание гарантировано —
**высота вывода типа**, или **операционная глубина SN**, или
введённое здесь множество `Red(τ)`.

Тейтовский подход — операционная семантическая интерпретация:
тип отождествляется с предикатом «терм ведёт себя предсказуемо».
Доказательство сводит SN к свойствам Red(·), которые сами
определяются индукцией по типу.

### 7.2. Альтернативные методы

* **Метод Жирана (reducibility candidates) для System F.** Тот же
  каркас, расширенный квантификацией: `Red(∀X. τ)` — пересечение
  по выбору интерпретаций для `X`. Технически сложнее
  (имперэкативное определение), но результат — SN для F.
  Исторически: одно из ключевых обоснований конструктивной
  теории доказательств.
* **Метод Гёделя «Dialectica».** Преобразование терма в
  не-типизированную форму с явным контролем рекурсии; SN
  доказывается через интерпретируемость в HA^ω.
* **Биорторгональность (Krivine).** Двойственное определение:
  Red(τ) и Red⊥(τ), биорторгональное замыкание. Технически
  изящно, обобщается до классической логики.

### 7.3. Что доказательство НЕ говорит

* Оно даёт только существование нормальной формы; **число
  шагов** редукции может быть **non-elementary** в размере
  терма (Statman, Mairson). STLC останавливается, но не
  «эффективно» в смысле TIME-сложности — это удивительный
  негативный результат, контрастный с интуицией о «ручной
  типизации».
* Оно не даёт полугруппового свойства для произвольных
  редукционных стратегий вне β. Если добавить η-редукцию,
  лемма CR2 требует усиления (η сохраняет приводимость
  тривиально, но взаимодействие с β нужно учитывать).

## 8. Проверка статусом одного шага

Терм:

```
ω = λx:o. x x        (где x: o, требуется o = o → ?)
```

**не типизируем** в STLC: применение `x x` требует `x : σ → ρ`
и одновременно `x : σ`, то есть `σ = σ → ρ`. Уравнение
**occurs-check** не имеет решения в индуктивных типах. Поэтому
терма

```
Ω = (λx. x x)(λx. x x)
```

в STLC **не существует**. Расходимость — невозможна по
синтаксической причине. Это и есть содержательная сила
типовой системы: уравнения, делающие нетерминирующее
поведение возможным, отсекаются на стадии типизации.

---

## 9. Список используемых фактов без доказательства

* Capture-avoiding substitution: `(λx. M)[y := N] = λx. M[y := N]`,
  если `x ≠ y`, `x ∉ FV(N)` (можно гарантировать α-переименованием).
* Перестановка β-шага и подстановки: `M →β M'` ⇒
  `M[x := N] →β* M'[x := N]` (стандартная лемма о β и
  подстановке).
* Принцип индукции по фундированному отношению: для отношения
  `R` без бесконечно убывающих цепочек, для свойства `P`
  достаточно показать «`P` для `a`, если `P` для всех `b` с `a R b`».
* Лексикографическое произведение фундированных отношений
  фундировано.

## 10. Подпись

Утверждение: каждый закрытый просто-типизированный λ-терм
останавливается. Доказано выше. Ничего не утверждается ни о
не-типизированном λ-исчислении, ни о расширениях с
полиморфизмом, ни о теории типов с зависимыми типами — там
аналогичные теоремы существуют, но с собственными
доказательствами.

Дата математики: ≈ 1967 (Tait), уточнение 1972 (Girard для F).
Дата записи: настоящая.
