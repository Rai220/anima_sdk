Reading.
  (1) MAIN_GOAL.md and AGENTS.md are read once per substantive session;
      CHANGELOG.md about half the time. The harness writes nothing into
      the conversation other than MAIN_GOAL.md (run.sh:13), so every other
      Read above is an active choice the model made.

  (2) The essays the model wrote in sessions 1–14 are largely inert as
      memory: of 9 essay .md files, 7 have ≤1 reads in the remaining
      sessions, and 2 have zero. The longest one (против_тёрстона.md)
      has never been reread. As input to later sessions, those essays
      effectively do not exist; the only persistent memory in active use
      is MAIN_GOAL / AGENTS / CHANGELOG and the post-rewrite measurement
      .txt files (each read 2–5 times).

  (3) Search behaviour is near-zero: 1 Grep, 0 Glob across 31 sessions.
      The model navigates this 25-file directory by directly Read-ing
      paths it already knows (mostly via the initial `ls`), not by
      pattern search. With this many files this still works; it would
      not scale.
